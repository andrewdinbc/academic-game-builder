# Academic Game Builder

Create curriculum-aligned educational games in minutes using AI.

## Features

- 🎮 **Game Creation** - Build interactive games with AI assistance
- 📚 **Curriculum Aligned** - Games designed to meet learning objectives
- ⚡ **Complexity Scaling** - Adjust difficulty from beginner to advanced
- 🧠 **AI Powered** - Uses Claude AI for game logic generation
- 🚀 **Play Instantly** - Games playable immediately in browser
- 📊 **Track Progress** - Monitor student engagement

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: CSS (with Tailwind utility-style classes)
- **AI**: Anthropic Claude API
- **Storage**: In-memory (expandable to database)
